# react-tetris-game

---

## Quick Start

```sh
# Build
$ docker build -t react-tetris-game:latest .

# Run
$ docker run --name react-tetris-game -d -p 3000:3000 react-tetris-game:latest

# in browser, visit localhost:3000 to view
```
